# Next.js TypeScript Portfolio

A modern personal portfolio website built with **Next.js** and **TypeScript**, designed to showcase projects, experience, and blog with a clean and responsive UI.

---

## Overview

This portfolio was built to present my work as a developer actively building and shipping web applications.

Rather than a static showcase, it reflects:

- Real projects and hands-on experience
- Continuous improvement and iteration
- A development workflow enhanced by AI tools

---

## AI-Assisted Development

This project was developed using Cursor as a core development tool to improve speed, structure, and overall code quality.

Used Cursor’s AI features (Ask, Agent, Plan) to:
- Quickly scaffold components and structure layouts
- Debug issues and refine implementations in real time
- Explore alternative approaches before finalising solutions
  
Focused on improving UI development efficiency by:
- Reducing time spent on design decisions through rapid iteration
- Breaking down interfaces into clean, reusable components
- Maintaining consistency across pages and sections

Rather than relying blindly on AI-generated output, I treated Cursor as a collaborative tool.
All generated code was carefully reviewed to identify incorrect logic, unnecessary complexity, and over-engineering. I validated results, simplified implementations, and ensured alignment with the overall architecture.

I also guided the AI using task-specific and context-aware skill (e.g. /frontend-design), which helped produce more structured and relevant suggestions—especially when refining layout and visual structure.

This workflow allowed me to move quickly while maintaining control, balancing development speed with code quality and maintainability.

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/ssoyeonisgood/NextjsTypescriptPortfolio.git
cd NextjsTypescriptPortfolio
```
### 2. Install dependencies
```bash
npm install
```
### 3. Run development server
```bash
npm install
```
Open:
```
(http://localhost:3000)
```
