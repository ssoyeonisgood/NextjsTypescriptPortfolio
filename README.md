# Next.js TypeScript Portfolio

A modern personal portfolio website built with **Next.js** and **TypeScript**, designed to showcase projects, experience, and blog with a clean and responsive UI.

---

## Features

-  Fast performance with Next.js
-  Responsive and modern UI design
-  Projects showcase section
-  About / experience section
-  Contact section (form or links)
-  Clean component-based architecture

---

## 🛠️ Tech Stack

- Next.js
- TypeScript
- React
- Tailwind CSS
- Vercel (deployment)

---

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/NextjsTypescriptPortfolio.git
cd NextjsTypescriptPortfolio
```
### 2. Install dependencies
```bash
npm install
```
### 3. Run development server
```bash
npm install
```
Open:
```
(http://localhost:3000)
```
